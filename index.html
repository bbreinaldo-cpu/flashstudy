🔑 Chaves de API

Suas chaves ficam salvas separadamente por usuário neste dispositivo. Ninguém mais tem acesso. Após salvar, o app preenche automaticamente.

💎 Como obter uma chave GRÁTIS do Gemini (Google)
  1. Acesse aistudio.google.com/app/apikey
  2. Faça login com sua conta Google
  3. Clique em "Create API Key"
  4. Copie a chave gerada (começa com AIza...)
  5. Cole no campo 💎 Gemini abaixo e clique em Salvar
✅ Plano gratuito: 1.500 requests/dia com gemini-1.5-flash
